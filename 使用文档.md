# SnapTranslate CN 使用文档

截图即翻译。按下快捷键框选屏幕上的任意文字，软件自动识别并翻译成中文，弹窗显示结果。适用于看外文网页、文档、视频字幕、图片里的文字。

---

## 一、安装

1. 打开下载到的 `SnapTranslateCN-1.0.0.dmg`
2. 把「SnapTranslate CN」图标拖到「Applications（应用程序）」文件夹
3. 从启动台或应用程序文件夹双击打开

首次打开如果提示「无法验证开发者」，在图标上**右键 → 打开**，再点一次「打开」即可（只需这一次）。

打开后，软件不会出现在程序坞，而是常驻在屏幕右上角**菜单栏**，显示为一个「译」对话气泡图标。

---

## 二、首次设置

第一次启动会自动弹出设置窗口（之后可随时通过菜单栏图标右键 → 设置… 打开）。

按以下顺序配置：

1. **翻译服务**：选择 OpenAI（这是通用的兼容接口，支持下方多家大模型）
2. **服务商**：从下拉菜单选你要用的，比如 MiniMax、DeepSeek、Kimi、通义千问、智谱 GLM、OpenAI。选好后「API 地址」和「模型」会自动填好。
3. **模型**：可点右侧下拉箭头切换该服务商的其他模型
4. **API Key**：粘贴你在对应平台申请的密钥（⌘V 粘贴）
5. **目标语言**：默认简体中文，可改
6. 点 **测试连接**，显示「连接成功」说明配置可用
7. 点 **完成** 关闭窗口

> 设置会自动保存，无需手动保存按钮。

## 三、授予屏幕录制权限

第一次截图时，macOS 会要求「屏幕录制」权限（用于截取屏幕做文字识别）。

1. 弹窗点「打开系统设置」
2. 在 **隐私与安全性 → 屏幕录制** 中打开「SnapTranslate CN」的开关
3. **退出软件并重新打开**（权限才会生效）

这是 macOS 的安全机制，只需设置一次。

## 四、日常使用

### 开始翻译（三种方式任选）

- **按快捷键**：默认 `⇧⌘2`（Shift+Command+2）
- **左键单击**菜单栏的「译」图标
- 右键菜单栏图标 → 截图翻译

屏幕变暗后，按住鼠标拖动框选要翻译的区域，松手即可。按 `Esc` 取消。

### 查看结果

松手后稍等片刻，弹出结果窗口：

- 上方是**识别出的原文**
- 下方是**中文译文**
- **复制译文**：一键把翻译结果复制到剪贴板
- **重新翻译**：对同一段文字再翻译一次

窗口可随时关闭，不影响下次使用。

## 五、菜单栏功能

**右键单击**菜单栏「译」图标，可看到：

- **截图翻译**：手动触发一次截图
- **翻译记录…**：打开记录窗口（见下）
- **设置…**：修改服务商、API Key、快捷键等
- **许可证…**：查看试用/激活状态，输入购买的许可证 Key
- **退出**：完全关闭软件

---

## 六、设置项说明

- **截图快捷键**：点输入框后直接按下想要的组合键（必须含 ⌘/⌥/⌃/⇧），按 Esc 取消修改
- **开机自启**：开机后自动在后台启动
- **翻译完成后自动复制译文**：翻译一完成就自动复制，省去点按钮
- **记录模式**：开启后每次翻译自动存档，可在「翻译记录」窗口浏览

### 翻译记录

右键图标 → 翻译记录…，在窗口里可以：

- 浏览历史翻译（原文 + 译文）
- 单条复制
- 打开存档文件夹
- 一键清空全部记录

记录只保存在你自己的电脑上。

## 七、许可证

软件提供 **7 天免费全功能试用**。试用期结束后，截图时会提示需要激活：

1. 点弹窗里的「购买许可证」前往购买页
2. 购买后会收到一个许可证 Key
3. 回到软件，右键图标 → 许可证…，粘贴 Key 点「激活」
4. 激活后本设备永久可用

## 八、常见问题

**截图后提示「未识别到文字」**
框选区域里没有清晰文字，或文字太小太模糊。尽量框选清楚、对比度高的文字区域。

**提示「翻译服务返回错误」**
检查 API Key 是否填对、对应平台账户是否有余额。可在设置里点「测试连接」排查。

**按快捷键没反应**
确认软件正在运行（菜单栏有「译」图标）；快捷键可能与其他软件冲突，可在设置里换一个组合键。

**菜单栏看不到图标**
MacBook 刘海可能遮挡了图标，把鼠标移到刘海附近或关掉一些其他菜单栏图标。

**翻译的是长文章**
软件会自动分段翻译，长文请耐心等待几秒。

**想暂时停用**
右键图标 → 退出。下次双击应用图标重新打开即可。

## 九、隐私

截图在你的电脑本地用 Apple 系统能力识别文字，截图本身不上传。只有识别出的**文字**会发送给你配置的翻译服务以获取译文。API Key 和设置只保存在本机。详见隐私说明。
