# 隐私说明

SnapTranslate CN 重视你的隐私：

- 截图仅在本机处理。OCR 使用 Apple Vision 框架，完全离线，截图本身不会上传。
- 仅识别出的文字会发送给你自己配置的翻译服务（OpenAI / DeepL / Google），用于获取译文。数据处理方式以相应服务商的隐私政策为准。
- API Key 与设置仅保存在本机 `~/Library/Application Support/SnapTranslateCN/config.json`，不会发送给除翻译服务商之外的任何第三方。
- 截图历史为可选功能，默认关闭；开启后文件仅保存在本机，可随时删除。
- 本应用不含统计、埋点或任何遥测。

不希望文字离开本机时，请不要对敏感内容使用翻译功能。
